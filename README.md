# Microsoft 365 Migration Project - User/Admin Side Preparation and Support
This repository describes migration project that I assisted.
Tasks completed are preparation, backups, migration support, post migration, and troubleshoot. 

## Overview
The goal of project was to transition from POP to Exchange service and creating tenant using new MS 365 Business Premium Tenant. 
Project includes adding previous domain to new tenant, support users with backup preparation and DNS record changes.

## Responsibilities
- Submitted pre-migration enviroment document to migration engineer.
- Created and updated backup guide for Win/Mac users both English and Korean.
- Troubleshooted backup/export issues and support users through it. 
- With new tenant, created user accounts, assigned licenses.
- Configured group mailbox and meeting resources on Exchange admin center. 
- Approved enterprise application with admin consent. 
- Guided user for structure of Sharepoint, OneDrive, Teams for file share. 

## Contributions 
- Resolved POP to PST/OLM issue/failures by creating profiles and vaildating data integrity
- Supported post-migration applications to users regarding Teams, Outlook, Sharepoint, and MFA registeration. 
- With Outlook, troubleshoot on Mac to resolve troubles and sent guide. 
- To make stakeholders schedule, with migration engineer went through DNS record changes, domain setting with remote access.
